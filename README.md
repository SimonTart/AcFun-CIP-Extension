# AcFun-CIP-Extension
AcFun Comment Instrumentality Project Extension A站评论补全计划插件

## 简介
安装插件后可以恢复A站视频或文章中被删除的评论，不保证能100%的成功率，且只针对2018-07-01之后发表的评论

## 官网地址
http://acfun.trisolaries.com/

## 开发
```bash
npm run dev
```

## 打包
```bash
npm run build
```
